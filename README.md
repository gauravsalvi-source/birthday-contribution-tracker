# Birthday Contribution Record

A React app for tracking Prathamesh & Aniket's birthday contribution collection.

## Features

- Add, edit, delete, and search contributor records
- Track contribution, Cake & Sevpuri expense, and balance per person
- View total collected, total spent, remaining balance, and contributor count
- Import and export CSV
- Saves records in browser local storage

The app starts with the CSV data:

- Total collected: Rs. 1400
- Total spent: Rs. 1301
- Balance: Rs. 99

## Run locally

Install dependencies once:

```bash
npm install
```

Start the React dev server:

```bash
npm run dev
```

Open the URL shown in the terminal, usually:

```text
http://127.0.0.1:5173
```

You can also double-click `open-app.bat`; it installs dependencies if needed and starts the dev server.

## Build

```bash
npm run build
```
